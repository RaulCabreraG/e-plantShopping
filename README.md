# e-plantShopping

Project Name: e-plantShopping

This project's structure was forked from [this](https://github.com/oslabs-beta/e-plantShopping) project for the IBM Developer Course.

## Running the App

To run the app, you need to have Node.js and npm installed on your machine. Then, run the following commands:

```bash
npm install
npm run dev
```

This will run the project in localhost:3000

## Notes

- The app uses React, Redux, and Redux Toolkit for state management.
- The app doesn't include the checkout functionality.
